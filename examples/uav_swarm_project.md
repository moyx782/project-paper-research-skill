# Example Project: LLM-Agent UAV Swarm Robotics

Use the project-paper-research skill.

Project:
I am researching an LLM-Agent-based decentralized UAV swarm system for disaster search and no-network environments.

System idea:

- LLM agent parses human mission intent
- mission is converted into validated JSON schema
- distributed UAVs perform task allocation
- deterministic algorithms handle path planning, collision avoidance, and control
- LLM is used for high-level planning, not low-level motor control
- possible tools: ROS2, Gazebo/AirSim/Isaac Sim, MAVLink, DDS, MCP, CBBA-style allocation, RAG over maps and mission context

Search areas:

1. multi-UAV task allocation
2. decentralized swarm coordination
3. CBBA / auction-based task allocation
4. LLM agents for robotics
5. tool-using LLMs in embodied systems
6. human-swarm interaction
7. natural language mission planning for robots
8. ROS2 multi-robot simulation
9. disaster search and rescue UAVs
10. communication-constrained multi-agent systems
11. map-based mission planning
12. safe autonomy with LLM high-level planners

Output:
Create a paper matrix and Markdown literature review report.
