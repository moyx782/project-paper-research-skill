# Example Project: AI Note Application

Use the project-paper-research skill.

Project:
I am building a Markdown-native AI note application. The MVP includes:

- Milkdown-based WYSIWYG Markdown editor
- built-in Pi Agent sidebar
- plugin-based architecture
- plain Markdown as source of truth
- local-first file management later
- future plugins for OCR, ASR, handwriting recognition, RAG, and workflow automation

Research objective:
Collect academic papers and high-quality references that can support the design, implementation, and report writing of this project.

Focus areas:

1. AI-assisted note-taking
2. personal knowledge management
3. Markdown / structured document editing
4. human-AI collaborative writing
5. AI annotation systems
6. tool-using LLM agents
7. agent workflow automation
8. RAG over personal knowledge bases
9. local-first software
10. plugin architecture for editors
11. OCR / ASR / handwriting recognition for notes
12. educational technology and learning support systems

Output:
Create a paper matrix and Markdown literature review report.
